# User Documentation

## Installation.

 - Download the app-relese.apk file.
 - Move it on your Android Phone
 - Enable "Authorized Untrusted Apps" in your phone's settings.
 - Open the file with Android APK Manager and install the application.

## Connexion to Imgur
